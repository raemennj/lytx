Auto brightness shortcuts

iOS Shortcuts

  1. Shortcuts > Automation > + > App.
  2. Select "Driver Camera QR" and choose "Is Opened".
  3. Add Action > Set Brightness to 100%.
  4. Optional: create another automation for "Is Closed" to restore brightness.

Android automation

  1. Open an automation app (Modes and Routines, Tasker, MacroDroid).
  2. Create a trigger for when "Driver Camera QR" opens.
  3. Add action: set brightness to 100%.
  4. Optional: add a close rule to restore brightness.
Tap once to keep the screen awake
